 School Management System

 Overview
This project is a **School Management System** designed to manage daily school operations efficiently. It provides separate dashboards for Admin, Teacher, and Student with secure authentication and proper form validation.

 Features

 Authentication
 User Login & Signup system  
 Form validation for secure data entry  

 Admin Panel
 View all students and teachers  
 Manage classes  
 Monitor student fees  
 Filter students by class  

 Teacher Panel
 View assigned class students  
 Manage and view student marks  

 Student Panel
 Attempt quizzes  
 View classmates  
 View class teacher  
 Generate fee vouchers  

 Technologies Used
 HTML  
 CSS  
 JavaScript
